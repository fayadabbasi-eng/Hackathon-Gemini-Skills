# Cape Town Summit Hackathon: Ideation Hub

This repository is a specialized environment for rapid ideation, strategy refinement, and pitch preparation for the Cape Town Summit Hackathon. It leverages agentic workflows to harden ideas into actionable specs and compelling scripts.

## ⚡️ Quick Start

This project uses custom Gemini CLI commands to orchestrate the hackathon workflow.

1.  **Ideation:** Run `/div-conv` to start the "Survival of the Fittest" ideation process.
2.  **Pressure Test:** Run `/grill-me` to stress-test your selected plan.
3.  **Pitch Prep:** Run `/script-gen` to generate a 2-minute hackathon script.

## 🛠 Project Structure

-   `.gemini/commands/`: The "brains" of the repo—contains custom command logic.
-   `specs/`: Finalized project specifications (`spec-v1.md`, etc.).
-   `scripts/`: Generated pitch scripts and presentation notes.
-   `GEMINI.md`: Internal project mandates and detailed workflow context.

## 🚀 The Workflow

1.  **Divergence:** Brainstorm 10+ ideas using different "Thinking Lenses".
2.  **Clustering:** Map ideas on a Novelty vs. Feasibility grid.
3.  **Survival:** Pass the "Interactive Grill" to identify and mitigate fatal flaws.
4.  **Hardening:** Generate a surgical Markdown spec for builder agents.
5.  **Pitching:** Transform the spec into a modular, 4-scene script with specific "Vibes".
